# Deep Cuts Salon - Premium Grooming Homepage

A premium, responsive landing page for **Deep Cuts Salon** located in Canal Garden, Lahore, Pakistan. Built using modern web technologies to showcase grooming services, interior gallery, customer reviews, and brands carried.

---

## Tech Stack

- **Core**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

---

## Getting Started

### Prerequisites

- Node.js (v18.x or later)
- npm (v10.x or later)

### Installation

1. Clone the repository or extract the project files.
2. Open your terminal in the project directory.
3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to view the application.

### Production Build

To build the application for production:

```bash
npm run build
```

This compiles the static assets and prepares the application for high-performance hosting on platform environments (such as Vercel, Netlify, AWS, etc.).

---

## Project Structure

- `/app` - Layout and routing entry points.
- `/components/site` - Reusable UI sections (Hero, Services, About, Why Choose, Testimonials, Location, Footer, etc.).
- `/components/ui` - Primitive components (e.g. Button).
- `/lib` - Config data and utility helper logic.
- `/public` - Static image resources and assets.
